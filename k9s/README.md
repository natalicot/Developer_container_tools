### Get started
```sh
# make sure k9s is installed
k9s version
```

```sh
# get into your k9s terminal
k9s
```

```sh
# press on your keyboard:
:
```

```sh
# type in pods and hit enter
:pods ↩
# now you see all your running pods
```
